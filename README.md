# 📋 TaskManager Project

This is a full-stack task management application built with:

- **Frontend**: Next.js + TypeScript + Tailwind CSS
- **Backend**: Laravel 12 + PHP + MySQL
- **Infrastructure**: Docker + Docker Compose
